Careview

What if you need to find a carehome for a loved one, but you're afraid of the living conditions on the inside?
Careview is a web application that helps you find a carehome in your region, based on transparent reviews from residents.

This is the final project of my coding bootcamp at Le Wagon Berlin. It is the result of a two-week sprint in a group of four people.
As a project lead, my work included the following:

  • Pitch idea
  • Project management
  • Write user stories
  • Design user journey in Figma
  • Develop from scratch to production
