TreeAndB

As a first project during the coding bootcamp at Le Wagon, we were supposed to create a clone of Airbnb. I led a team of four people during this one-week sprint.
TreeAndB allows you to book an original getaway in nature as well as list your own getaway for rent.

The application features:

  • Create a profile
  • Search for a getaway
  • Book a getaway
  • Leave a review
  • Access personal dashboard
